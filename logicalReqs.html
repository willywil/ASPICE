Logical Parts

The logical parts of the system are described below:

  1. State Machine Spec
    • Identify Transitions/Triggers
  2. Interactions Spec
    • Identify Communication Processes
  3. Signals List
    • Identify Protocols Needed
    • Identify Speeds Needed
  4. Interfaces
    • Identify the to-from for each signal
  5. Logical Parts List
    • Identify (at an abstract level) what needs to be inside the system
  6. Block Diagrams
    • Graphically describe the sub-system using visual description
  7. System-Level Measures of Effectiveness (MOEs)
    • Identify the criteria you plan to use to determine how well the system works
  8. Add Performance Requirements
    • Based on MOEs
